# ASCII Mockup Plugin Marketplace

A Claude Desktop CoWork plugin marketplace containing the `ascii-mockup` plugin — for creating ASCII wireframes, dashboards, system diagrams, data schemas, and slide deck layouts before building anything.

## Install in Claude Desktop

1. Open Claude Desktop → Settings → Plugins
2. Click **Add marketplace from GitHub**
3. Enter: `https://github.com/ccdsky/ascii-mockup-plugin`
4. Install the `ascii-mockup` plugin

## Plugins

| Plugin | Description |
|--------|-------------|
| [ascii-mockup](./ascii-mockup) | ASCII wireframes, dashboards, diagrams, schemas, and slide layouts |
